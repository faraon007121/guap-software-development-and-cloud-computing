####
Необходимая версия Python: 3.12
####
Перед запуском скриптов установить виртуальное окружение и необходимые зависимости, указанные в requirements.txt
Сделать это можно средствами вашей IDE, либо выполнив следующие команды в корневой папке проекта:
pip install -r requirements.txt
Windows:
python -m venv venv
venv\Scripts\activate
macOS/Linux:
python3 -m venv venv
source venv/bin/activate
